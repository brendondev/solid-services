# Templates de Importação

Esta pasta contém templates prontos para importação de dados no Solid Service.

## Arquivos Disponíveis

- `clientes.csv` - Template para importação de clientes com exemplos
- `servicos.csv` - Template para importação de serviços com exemplos
- `fornecedores.csv` - Template para importação de fornecedores com exemplos
- `importar-tudo.csv` - Template para importação de múltiplas entidades de uma vez

## Como Usar

1. Baixe o template desejado
2. Abra no Excel, Google Sheets ou editor de texto
3. **Mantenha a primeira linha (cabeçalho) intacta**
4. Substitua os dados de exemplo pelos seus dados reais
5. Salve o arquivo
6. Importe via interface web em `/dashboard/import`

## Formato

- **Encoding:** UTF-8
- **Separador:** Vírgula (,)
- **Formato:** CSV padrão

## Observações

- Não altere os nomes das colunas
- Você pode adicionar mais linhas de dados
- Mantenha o formato dos dados conforme os exemplos
- Para mais informações, consulte a [documentação completa](../../../../docs/IMPORTACAO-DADOS.md)

## URLs de Acesso Direto
